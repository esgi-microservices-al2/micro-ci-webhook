# Micro CI - API Webhook


## I. Lancement de l'API

## A. Pré-requis

### 1. Variables d'environnement

Settez les variables d'environnement dans votre terminal :

`set BROKER_URL=127.0.0.1` - IP du Broker

### 2. Dépendances

Installer les dépendances avec :

`npm install`

##B. Lancement

Lancer en mode "Dév":

`npm run dev`
